# 段永平：雅虎投资分析与价值投资理念

> "很典型的美国投资人的看法，居然根本不提淘宝和支付宝雅虎的微妙资产"

## 雅虎投资分析

**投资现状与策略**

段永平表示一直在买入雅虎股票：
- "每个月股权到期日后我就可能会有些钱买"
- "买yahoo风险挺大的，估计一时半会不会涨，最好不要跟"

**风险认知差异**

- 对普通投资者："我觉得不懂得人买的话风险很大"
- 对自己而言："对我来讲，yahoo目前是风险最小的"

**估值与前景**

- 市场分析："如果一切都非常理想的情况下，yahoo到40+倒是有可能的，但至少也要3年才有可能看得到吧"
- 公司评价："客观地讲，雅虎本身现在还不能称为好公司，但新CEO上来后，yahoo将有机会成为一个不错的公司"

## 投资哲学与理念

**巴菲特思想的影响**

> "巴菲特'一生只需富一次的意思'是不要冒不该冒的险，不然可能需要去想办法富第二次。不用margin也是这个意思。"

**投资心态**

- "投资不敢认为一切都是理想的"
- "一定要知道自己为什么投"
- "没有时间长的准备是不应该投资的，早晚会掉进去"

## 历史投资案例回顾

- 市值：2000多万（不到3000万）

**关于标王的澄清**

## 投资分类方法论

网友提出的企业投资类型分类框架：

1. **缓慢增长型 + 快速增长型**：转机股
2. **缓慢增长型 + 稳定增长型**：便宜的公司  
3. **稳定增长型 + 快速增长型**：贵州茅台、可口可乐等
4. **周期循环型 + 隐蔽资产型**：中国石油、北伯林顿
5. **困境反转型 + 周期循环型**
6. **困境反转型 + 隐蔽资产型**：圆石滩
7. **好公司（稳定或缓慢） + 隐蔽资产型**：雅虎
8. **好公司（稳定或缓慢） + 困境反转型**：2001年网易、美国运通
9. **好公司（稳定或缓慢） + 周期循环型**

1. **理性投资**："看东西一定要理性，不能一害怕就乱了"
2. **长期视角**："投资不敢认为一切都是理想的"
