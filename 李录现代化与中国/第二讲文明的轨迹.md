# 新史学家、斯坦福大学教授Ian Morris的社会发展指数

新史学家、斯坦福大学教授Ian Morris提供了定量记录人类长期文明历史轨迹的计量方式，他把这种计量方式叫做**社会发展指数**，即一个社会能够办成事的能力。

## 一、社会发展指数的基本原理

社会由人组成，同为动物的人需要消耗能量。根据能量守恒的原理，一个社会要能办成事，主要需要摄取和使用能量的能力。所以要想衡量社会发展的程度，最重要的指数就是摄取能量和使用能量的能力。

下面从**计量内容**、**计量方法**、**计量对象**这三个方面解释这种计量方式。

## 二、计量内容：四个核心维度

Morris把一个社会能够摄取能量和使用能量的能力分成四个方面：

1. **摄取能量的能力**
   - 主要指社会中的每一员每一天能够摄取的食物、燃料和原材料的能力

2. **社会组织的能力**
   - 定义为在一个社会里最大的永久性居住单位的人数
   - 在相当长一段时间里也就是最大城市的人口数
   - 人口越多，对社会组织的能力需求就越高

3. **信息技术的能力**
   - 社会组织的成员每天都需要交流、储藏、记忆各种各样的信息
   - 信息技术是人类使用能量的最重要方式之一

4. **战争动员的能力**
   - 战争作为人类消耗能量的重要来源更不必赘述

> 这四个方面并非人类活动的全部，却是最具有代表性的人类摄取和使用能量的方式。更关键的是这四个标准能够在一切社会中横向比较，也可以在很长的时间范围里纵向比较。

## 三、计量方法：指数化处理

在考虑计量方法时，Morris选择了指数的方法：

- **时间跨度**：公元前14，000年到公元2000年
- **总分设定**：公元2000年的数值为1000分
- **分项配分**：四个方面平均分配，每项满分250分
- **基准标准**：用公元2000年代表东西方最高水平的每项社会发展指数作为满分

### 具体示例

- **美国**（西方最发达地区）：平均每人每日能量摄取约228，000大卡 → 250分
- **日本**（东方最发达地区）：平均每日每人能量摄取约104，000大卡 → 114分

### 时间间隔设置

由于历史早期数据获取困难，Morris采用了不同的时间间隔：

| 时间段 | 数据采集间隔 |
|--------|-------------|
| 公元前14，000年 - 公元前4000年 | 每一千年 |
| 公元前4000年 - 公元前2500年 | 每五百年 |
| 公元前2500年 - 公元前1500年 | 每二百五十年 |
| 公元前1500年 - 公元2000年 | 每一百年 |
| 现代以后 | 每年甚至每月 |

> 要对一万六千年的数据都进行比较精确地估算，就需要考古学、气象学、物理学、生物学在过去几十年取得的成果辅助。

## 四、计量对象：东西方文明中心

Morris把测量对象定义为：公元前9600年以后欧亚大陆上农业文明形成时出现的两大文明中心，以及此后传承这两大中心的各个文明中心。

### 西方文明中心变迁
```
两河流域和约旦河附近的侧翼丘陵区 → 
美索不达米亚、叙利亚、埃及、地中海、罗马 → 
巴尔干半岛 → 
地中海、南欧、西欧 → 
美国
```

### 东方文明中心变迁
```
黄河流域 → 
黄河与长江冲积平原中间 → 
长江流域 → 
中国东南沿海和日本（二十世纪后）→ 
日本（公元2000年左右代表）
```

## 五、考古学支撑

史前的记录有很多数据需要估算，因此考古发现是重要的信息来源：

- **1870年后**：开始使用地层学研究（stratigraphy）
- **1950年后**：开始使用放射性碳定年法（Carbon Dating），带来实质性的飞跃
- **1970年代后**：逐渐形成系统的史前记录知识体系

## 六、东西方社会发展轨迹

Morris和他的团队通过大量的工作，将人类社会发展的指数绘制成一系列图表，这些图表有助于我们直观了解东西方社会发展的历史轨迹。

### 图1 东西方社会发展指数（公元前14000年－公元2000年）
*来源：Ian Morris "Social Development （2010）"*

从图1可以看到：
- 一直到公元前3000年左右，东西方的发展几乎看不出任何差别
- 之后虽然两方的发展曲线都发生了一些变化，但仍然非常缓慢
- **公元1800年以后**，整个图表发生了根本性的变化，社会发展轨迹像坐了火箭一样，呈现出飞跃式发展

### 图2 东西方社会发展指数线性对数模型（公元前14000年－公元2000年）
*来源：Ian Morris "Social Development （2010）"*

通过对数处理，可以把东西方的比较看得更清楚一些。

### 图3 东西方社会发展指数（公元前1600年－公元1900年）
*来源：Ian Morris "Why the west rules – for now (2011)" P168*

这个图表体现的历史是文字记载相对清晰、人们比较熟悉的一部分历史。

## 七、历史轨迹总结

结合图2、图3，可以看到：

1. **公元前14000年左右到公元500年左右**：西方一直领先东方
2. **公元541年左右到1773年左右**：东方领先西方，在这一千二百多年里保持领先地位
3. **公元1800年以后**：西方不仅追上了东方，而且率先进入飞速发展期，把东西方之间的差异扩大成对全球的统治
4. **二十世纪开始**：东方的社会发展指数开始起飞，今天虽然仍然大大落后于西方，但是已经显示出能够追上西方的迹象

这就是一万六千年的人类文明在东西方两地的轨迹。

## 八、后续展望

本系列之后几篇将重点解释：

- 人类文明轨迹的成因
- 东西方在文明发展过程中的异同
- 公元1800年以后社会呈现火箭式飞速发展的原因
- 东西方的比较，进一步解释西方为什么能够在近代统治世界
- 解释中国在近代的落后

只有在理解历史轨迹和成因的基础上，才能回答今天中国如何能够赶上西方，总结中国现代化的特性，展望东西方未来。
