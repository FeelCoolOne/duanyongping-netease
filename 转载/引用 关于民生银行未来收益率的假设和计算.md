# 关于民生银行未来收益率的假设和计算

> 本文转载自潮·慢·ing《关于民生银行未来收益率的假设和计算》

## 基本假设与预测

### 核心假设
1. **估值不变**：按照专家们的分析，未来五年银行股估值保持在5倍市盈率（已体现所有不利利空因素）
2. **盈利增速递减**：民生银行盈利增速逐步下降，未来四年分别保持40%、30%、25%、20%的增长

### 预测数据
| 年份 | 每股收益（元） | 分红（元） |
|------|---------------|-----------|
| 2012 | 1.37          | -         |
| 2013 | 1.78          | 0.41      |
| 2014 | 2.23          | 0.53      |
| 2015 | 2.67          | 0.67      |
| 2016中期 | -       | 0.80      |

### 投资收益分析
- **分红合计**：2.71元（2013-2016中期）
- **2016中期股价**：13.35元（按2.67元×5倍PE计算）
- **实际价值**：16.06元（含已分红利）
- **4年总收益**：约10元/股
- **总收益率**：约166%
- **年化收益率**：约41.5%

### 结论
民生银行有实实在在的业绩支撑，13.35元股价仍处低位，不存在高估问题，投资者无需担心大幅下跌风险。

## 专家观点与讨论

### 段永平观点

> **关于盈利增速**：我不太明白这个保守的盈利增速下降是什么意思，为什么不可能亏钱呢？很多人搞不清楚物理学里面的速度和加速度的区别，老是把物理里的加速度当成速度。要知道，加速度是可以瞬间变负数的。

> **关于投资价值**：本人不懂银行，不知道为什么别人会得出这个结论。但是，如果这个结论是对的话，那这个银行确实不贵。

> **关于风险认知**：先不谈风险的投资能叫投资么？

> **关于投资理念**：投资要理性。投资里利益之上的追求大概主要是指将所得回馈给社会吧。投资的信仰指的是：相信长期而言股市是称重机，对没有信仰的人而言，股市永远是投票器。

> **关于银行本质**：重置是不会拿着烂帐开始重置的。如果你能知道烂帐有多少，就知道重置的成本。我不懂银行。对我而言，即使是让我开银行我也不愿开的。

### 反对观点分析

**主要质疑**：
1. **高息差问题**：中国银行股高利润来源于高额息差（占比80%以上），这种现象绝对不合理，一定会改变
2. **坏账风险**：银行股目前5倍PE，因为面临巨大危机，经济基本面出现重大危机
3. **投资风险**：银行自身投资业务不稳赚，只有息差稳赚（国家特色，不具备长期性）
4. **竞争冲击**：充分竞争环境下，与欧美大银行相比优势不明显

### 支持观点分析

**积极因素**：
1. **政策保障**：政府不能也不会制定对银行不利的政策，政府是银行实际控制人
2. **资产质量**：地方债务是优质资产，有土地抵押和还款来源
3. **风险控制**：民生银行不良贷款率连续10年维持在1.5%以下
4. **业务特色**：商贷通不良率仅0.14%，利差达8%，双重担保机制
5. **转型优势**：主动转向小微贷款，避开政策性贷款任务

### 投资建议总结

**谨慎派**：认为看不懂银行，不敢下手，等待更好机会

**乐观派**：认为民生银行潜在风险可能比所有国内银行都低，当前价格很吸引

**中间派**：只能买最保守的银行，合适价格买入合适银行是可以的
