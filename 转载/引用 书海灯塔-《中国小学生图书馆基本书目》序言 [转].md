# 书海灯塔
## ——《中国小学图书馆图书基本配备书目》序言

### 深圳市天图教育基金会

> 我心中常默默念想，天堂应该是图书馆的模样。
> 
> ——诺贝尔文学奖获得者阿根廷国家图书馆馆长 博尔赫斯

"天堂应该是图书馆的模样。"这是文学大师博尔赫斯的一句诗。这句诗也可以反过来说，"图书馆应该是天堂的模样"。

可是，如果让一个中国小学生谈起他们的学校图书馆，他们会得出怎样的印象呢？

请看看一项调查中孩子对图书馆的印象：

**问：你喜欢到学校的图书馆借书吗？（约63%的学生不喜欢。）**  
原因：  
- 里面的书不好看，喜欢看的借不到  
- 推荐的新书太少了  
- 图书管理员不让我们自己找，我又不知道哪些书是我喜欢看的  

**问：你喜欢在学校图书馆里读书吗？（约70%的学生不喜欢。）**  
原因：  
- 在图书馆里读书不太舒服，桌椅高度不合适  
- 太吵，没有合适的位置坐着读  
- 书的摆放不合理，我们不好找  

**问：你喜欢怎样的图书室？（摘录部分学生的回答）**  
- 书柜不要太高，方便我们找书  
- 最好木地板，上面有垫子，可以坐着读，也可以躺着读  
- 如果安放一些布袋椅就好了  
- 管图书的老师要亲切些  
- 我们可以在图书馆里上阅读课就好了  
- 开放的时间多一点，我们可以经常进来  

这是来自广东的一份调查。如果放到全国来调查，情况会更糟糕。在全国，很多农村小学的学生还不知道"图书馆"为何物，他们除了课本几乎没有读过其他图书；而城市中很多学校的图书馆也处于学校边缘位置，图书馆的主要作用是为了应付上级检查。

这，正是我们想改变的。

## 一、为什么要研制这个书目

### （一）深圳市天图教育基金会的使命

天图教育基金会是一家助学育人、改善贫困地区教育资源匮乏状况、引导社会关注农村教育的基金会。2010年11月正式通过深圳市民政局审批。

基金会把对乡村儿童的阅读服务作为工作重点，其理念是"阅读带我飞，携手共播爱与知"，其使命是"致力于让中国乡村儿童热爱阅读并成为终身阅读者"。

同大部分基金会一样，天图教育基金会为学校捐书时，首先面临的就是书的选择问题。哪些书最适合小学生阅读？不同年级的小学生阅读需求有什么不同？如何才能买到挑选出来的好书？

解决这个问题的难度，远远超过想象。在最初阶段，基金会的工作人员边干边学，收集大量有用信息，阅读大量图书。后来，天图教育基金会接触了这个领域的专业人士，并邀请一些知名儿童阅读专家加盟天图教育基金会。

因为这些专家的参与，天图教育基金会捐赠图书的质量越来越高。由此，基金会设想，如果能有一份为中国小学图书馆研制的书目，那些热衷于推广阅读的学校、组织机构和个人就犹如拥有一份地图，按图索骥就能找到那些优秀童书。

在儿童阅读上，城乡的孩子没有本质不同，每个孩子都喜欢阅读优秀的童书。

### （二）为小学生选择最优秀的图书

学校图书馆的图书质量决定了孩子的阅读品位，直接影响孩子的阅读兴趣。可是，好书进入学校困难重重，究其原因，大致如下：

**第一、思想不重视。** 在大多数学校图书馆都是摆设的状况下，有没有好书就无所谓了；在整个教育对阅读能力轻视的状况下，大部分家长、校长和学生都主动或被动地把精力投到了英语和"奥数"上；教师受升学压力的影响，也不敢放开让学生到图书馆遨游，只能让学生在教室里家里题海潜水；至于学生，则很少有人告诉他们阅读的美妙，也很少有人引导他们分享图书馆里的收获，只有任电视和电脑游戏侵蚀他们可怜的课外时间。

**第二、资金不充足。** 在教育经费紧张的情况下，又有人均图书量的考核指标，学校就只有想办法"少花钱，多买书"，牺牲图书品质去达到数量。考核标准的不科学直接导致图书采购流程和采用采购书目的不科学，配到学校的往往都是价高质次的书，出版社在纸张、印刷上偷工减料，在内容上东拼西凑。有些图书甚至是论斤买回来，图书内容没有保证。

**第三、选书不专业。** 给孩子选择适合他们阅读的书，是一件专业的事。从版式的设计、插图的搭配，到主题的适切、语言的适宜，需要长期进行童书研究的专业人士来帮助。但大部分学校缺少专业能力进行这方面的专业判断，大部分学校也没有时间去审读图书。

好书进入到学校困难重重，我们希望能通过这个书目，解决选书上的一些问题，把那些最适合孩子阅读的优秀图书筛选出来，让重视阅读的局长、校长、老师或家长们，一起努力，让优秀图书伴随他们成长，让阅读带他们飞翔。

### （三）国内图书出版的良莠不齐

我国是图书出版的大国，每年有30多万种图书出版。我国也是童书出版的大国，每年有4万多种童书出版，网上书城的童书品种有22万多种，而一般学校配备的书不过几千种。

可见，必须要花大量时间对现有图书进行筛选、比较。例如，国内出版的童书重复现象很严重，安徒生童话、格林童话有上百种，那就要比较翻译的质量如何，改编的质量如何。

为了保证采购图书的资金使用效益的最大化，在确保图书整体质量的前提下，想买到更多的书，要靠判断图书的性价比来实现。例如，图画书是孩子阅读的必需种类，但图画书的价格普遍偏贵，动辄三四十元。但是，有很多平装图画书品质也很好，价格却便宜很多。所以，我们就尽量选择平装图画书。当然，对一些只有精装本的经典图画书，我们也选到书目中，因为孩子在阅读初期，需要读到最美好的作品。

质量和价格有时是矛盾的，我们在面临这样的选择时，总是把质量放在首位。

### （四）不同学校对童书的需求不同

虽然国家规定的小学图书馆的配备标准为人均15—30册，但是各个学校的情况也非常复杂，需求也不尽相同。有的乡村小学只有几个班几十个人，图书配备量不大；有的城市小学规模大、人数多，图书配备量要加大。

针对这种情况，我们按照人均10本的比例，做了三个书目，分别为：一级书目、二级书目、三级书目，另加一个教师书目。数量分别是900册、1100册、1200册、70册。

| 书目级别 | 数量 | 颜色标识 |
|---------|------|----------|
| 一级书目 | 900册 | 浅橘色 |
| 二级书目 | 1100册 | 浅蓝色 |
| 三级书目 | 1200册 | 浅绿色 |
| 教师书目 | 70册 | 浅粉色 |

### （五）改变教师的阅读

天图教育基金会认为，阅读的问题是教育观念的问题，教育观念的问题依赖教师的阅读。只有教师是爱阅读的人，学生的阅读才有希望。

因此，我们在本书目中特意附了"教师阅读类"图书70册。这70册图书主要分为两类：

**第一类是"阅读指导类"图书**，主要是改变阅读理念，传授阅读方法；

**第二类是"教师专业成长类"图书**。在这类图书中，我们没有选一些理论艰涩的教育类图书，而是关注教师作为个体的"人"，改变教师的职业倦怠、增进教师的职业神圣感、改善教师的教学方法、关心教师的健康、拓展教师的知识面。

我们相信，阅读同样是教师专业化成长的最佳途径。同时，我们真诚地建议，教师可以多阅读一些童书，因为优秀的童书同样能帮教师探寻童年的秘密，理解教育的真谛。

## 二、书目的研制理念是什么

古今中外，有不少或机构都推出过书目，这些书目由于研制书目的推荐主体的文化立场、阅读经验、历史环境不同，每一个书目都有不同的研制理念。

我们希望《中国小学图书馆基本配备书目》不只是一个好书目录，还是一个有内在联系、有教育理念、有思想深度的书目。因为我们相信图书馆是更大的学校，学生可以通过阅读实现自我学习、自我成长。

具体下来，我们有如下研制理念：

### （一）适切性

图书浩如烟海，挑选出适合学生阅读的好书并不容易。有的人认为阅读是私人的事情，个人趣味不同，没必要窄化孩子的阅读，还有人认为"喜欢读什么书，孩子说了算"。

这些说法都似是而非。大人是孩子阅读的"守门人"，孩子的阅读趣味要靠大人来引导，年龄越小越需要指导。或许孩子的阅读兴趣不同，但不可否认适合孩子阅读的好书有基本的标准，好书的价值不可替代。

什么样的作品才算是适合孩子阅读的好书呢？法国著名文学史家保罗·亚哲尔在《书·成人·儿童》一书中，曾把适合儿童阅读的好书的标准定为以下几类：

- 忠实于艺术的书。就是诉之于"直观"，而得以培养儿童观察力的书。是孩子们读了，也会觉得它具有质朴之美的书。
- 可以解放儿童的心，使他们喜悦的书。这种书可以保护儿童，守住想象世界的幸福，避开现实法则的束缚。
- 能把人类高贵的感情吹进儿童心灵的书，使儿童尊重一切生命——包括动物的生命、植物的生命、森罗万象的生命。
- 承认游戏是重要的，不可或缺的活动的书。知性和理性的锻炼，并不是可以立即产生利益的，也不是能在实际生活中发生作用的。好的书应该不以立即的效果为目的，而且决不可把它当做目的。
- 启发儿童知识的书，帮助孩子认识人性——人类心情的书。它们能促进孩子们旺盛的成长力，使他们的精神圆熟，绽开睿智的花朵。
- 道德的书。这种道德是永远不变的真理，能让人类的心灵活泼起来、激奋起来，愿奔向真理的道德。

当然，这个标准还显得比较"虚"，这几类书并没有囊括所有好书，但它给我们的启发是：大人的审美判断能力是选择优秀童书的关键。所以，我们在这个书目的后面，还推荐给老师们一些儿童阅读指导方面的书，希望他们能重视儿童阅读，也能建立起好书的标准。

### （二）经典性

好书，只是这个书目最基本的要求，但仅仅是好书远远不够，这些书还应该具备经典性。

对于何谓经典，不同的人有不同解释。意大利文学家卡尔维诺曾经用15条文字来定义经典，如：

1. 经典是我们常听人说，"我在重读……"而不是"我在阅读……"的那类书。
2. 我们将人们读了爱不释手，加以珍藏的书冠之以经典；但并非只是那些有幸初次阅读它们的人，才精心珍藏它们，欣赏它们。
3. 经典具有特异的影响力，它们不可能从头脑中清除，它们潜藏在大脑的记忆层中，披上了集体或个体无意识的伪装，等等。

经典是能够打败时间的书，是被一代又一代人所记忆的书。

儿童文学在几百年的发展历程中，形成了自己独特的一些经典作品。这些作品跨越时间和空间，被世界上不同国家的孩子阅读。例如，《好饿的毛毛虫》《爱丽丝漫游奇镜》《秘密花园》《长袜子皮皮》。

世界各国都有一些儿童文学奖项，这些奖项为我们寻找经典提供了方便，例如，美国的纽伯瑞儿童文学奖、凯迪克图画书奖、国际安徒生奖、加拿大的总督文学奖、中国的全国优秀儿童文学奖。

儿童文学的经典可能不像成人文学的经典那样被公众熟知，但更加能够照顾儿童的阅读兴趣。这样的经典作品即使是放到整个文学的天平上，也毫不逊色。

学生在阅读经典作品的过程中，会逐步培养起阅读的品位和眼光，再去阅读其他图书时，会有"一览众山小"的感觉。

### （三）趣味性

很多大人抱怨，孩子只喜欢看电视、玩电脑游戏，不喜欢看书。究其根本原因，是推荐给孩子阅读的书趣味不够。

让孩子阅读趣味性差的书，会破坏他们的阅读胃口，甚至会把读书作为一种"惩罚"。

对儿童文学的经典作品而言，趣味性本来就是其重要特征。趣味性也是让孩子爱上阅读、亲近书籍的重要力量。看似简单的图画书，其实正是文和图的合奏，带给孩子不同的阅读感受。例如，安东尼·布朗的《我爸爸》，在画面中藏了许多有趣细节。

同时，我们也应该注意到，孩子的趣味和成人的趣味是不同的。成人感受到的趣味孩子可能体会不到，而孩子喜欢的趣味成人又无法理解。这其中最突出的就是卡通读物。

很多大人认为卡通图书对孩子是有害的，或者是无营养的，甚至无法理解为什么孩子如此喜欢卡通动漫。

在现代社会中，大人应该以宽容而开放的心态来看待孩子的阅读，其实，许多卡通书都是有益、有趣的，只有有色情、暴力的卡通书（色情和暴力也并非卡通所独具）对孩子才是有害的。

1990年、1991年来自32个国家二十多万名儿童参加国际教育成果评估协会(IEA，the International Association for the Evaluation Achievement)的考试，结果芬兰儿童的阅读分数最高，而芬兰儿童最普遍的消遣书刊是什么呢？59%的芬兰儿童几乎每天都看一本漫画书。

其实，卡通读物像武侠小说一样，对阅读的流畅度非常有帮助。大人应该避免以自己的阅读偏好来影响孩子，我们知道，在电视和网络无处不在的情况下，让孩子爱上阅读更加困难，如果没有趣味性作为"催化剂"，很难让孩子喜欢阅读。

所以，在这个书目中，我们为了照顾孩子的阅读兴趣，特地少量选入了卡通图书（如漫画书、武侠小说、侦探小说等）。

我们并不认为，这些书的选入会降低书目的质量，相反它是对孩子阅读兴趣的尊重。这些书甚至可能成为未来图书馆里借阅率最高的书。

### （四）科学性

不同年龄阶段的孩子有不同的阅读心理和阅读兴趣，这就要求对图书进行适当的年龄分级，在最适合的年龄阶段推荐给他们最适合阅读的书。

所以，我们按照小学低、中、高年级，学生的发展特征分别给学生推荐图书。

| 年级段 | 发展特征 | 阅读特点 |
|--------|----------|----------|
| 低年级 | 形象思维为主，识字量有限 | 以图画书为主，图文并茂 |
| 中年级 | 开始向抽象思维过渡 | 图文书与文字书并重 |
| 高年级 | 抽象思维发展，理解力增强 | 以文字书为主，适量配图 |

当然，这种划分也不是僵化的，每个孩子由于阅读基础和生活经验的不同，理解一本书的程度也不同。我们在书目中呈现的小学低、中、高年级的图书，也是相对的，老师可以按照学生的实际阅读能力进行调整。

### （五）均衡性

我们相信，阅读对人精神的影响是深广而持久的。所以，这个书目不只是研究书，还研究人，研究孩子在成长过程中必备的"心灵维生素"。

这是我们在书目研制过程中最感困难的地方，也是书目最有价值的地方。

我们按照"科学、人文、文学"三大类来选择图书。我们希望小学生通过这三类图书的阅读，能达到阅读的"营养均衡"。

科学、人文和文学作品，实际上是表现"真""善""美"的有效文本形式：

- **科学**是研究各种现象的本质和规律的知识体系，是反映事实真相的学说。其重在追求"真"。
- **人文**是文化生活中与人关系非常密切的先进部分和核心部分，主要体现在各种艺术样式以及先进的价值观及其规范，是重视、尊重、关心、爱护人的文化。其重在追求"善"。
- **文学**是描写社会生活和心理活动的语言文字的艺术，其重在追求"美"。

#### 真善美的核心价值维度

**横向维度**包括人与现实世界的关系和人与优秀文明成果两大方面：

从人与现实世界的关系维度来看，主要有八个价值领域：
- 人与自我 → 人生观
- 人与家庭 → 家庭观  
- 人与社会 → 群己观
- 人与国家 → 国族观
- 人与自然 → 自然观
- 人与世界 → 国际观
- 人与历史 → 历史观
- 人与未来 → 宗教观

**纵向维度**包括四个方面：

1. **人应该追求的品质与美德**：爱心、善良；乐观、积极；勤奋、勤劳；正直、公正；勇敢、坚强；自信、自立；负责、责任；宽容、宽恕；真诚、守信；俭朴、节约；自然、健康；信念、理想等

2. **人应该具有的态度与作风**：爱憎分明、懂得感恩；朴素、美丽；文明、淑雅；幽默、风趣；平易、谦虚；反思、自律；坦然、无畏；尊重、敬畏；合作、团结；惜时、守时；坚持、专注；认真、努力；谨慎、细致；博学、智慧；淡泊、豁达；忍让、谦让；礼貌、守纪；廉洁、务实；观察、体认；想象、创新等

3. **人应该遵循的准则与秩序**：爱国、爱人；忠义、坚贞；友谊、友好；和平相处；卫生、环保；悲悯、信仰；纪律、规则；平等、民主；独立、自由；美好、幸福等

4. **人应该了解的科学知识及应具备的科学精神**：科学认知、科学理解；科学意识、科学思维；科学兴趣、科学想象；科学分析、科学思考；科学态度、科学精神；科学质疑、科学验证；科学模仿、科学发明；科学发现、科学创造等

我们沿着这样的教育目的和教育方向去选择图书，希望尽可能包括体现真善美以及八大价值领域的主要概念与关键内容。我们希望那些图书，能够编织出一张美丽的网，呵护孩子在漫长的旅途中保持着纯真、快乐与自信。

## 三、如何使用这份书目

这是一份蘸满爱意的书目。研制这份书目，是因为我们相信阅读的力量。我们想通过书籍，让孩子们感受生命的美丽与神奇；我们想通过书籍，让孩子们理解人生的丰富与诗意；我们想通过书籍，带孩子们到广阔光明的地方。

如果书是一片海洋的话，书目就犹如书海灯塔，照亮了那些在书海里遨游的孩子。

我们将天图教育基金会关于阅读的理想，转换成研制书目的理念，呈现给有志于改变中国小学图书馆现状的个人或组织。我们认为，这份书目同样适合城市学校和乡村学校作为图书馆的采购书目。

### 使用建议

**（一）按照书目的级别**
学校可以根据需要优先考虑配备一级书目的图书，当一级书目的种类无法满足学校需要的时候再配备二级书目的图书，然后是三级书目。图书数量达到800的图书馆配齐教师用书。

**（二）书目是一个基础**
不同的学校和机构可以进行第二次选择，例如，根据学校的规模、学生的阅读需求、学生的阅读基础、希望突出的阅读重点等等。如果3270种还无法满足学校的购书需求，可以增加复本量或新品种图书。

**（三）家庭社区指导**
书目也可以看做家庭、社区为儿童购置图书的指导手册。

**（四）分类采购**
书目按照"文学、科学、人文"分为三大类。为了方便您使用书目，在每个大类后面都标注了细分小类，小类为您选购图书提供进一步参考。我们在排列书目时，按照出版社来排列，方便您集中采购。

**（五）市场缺货处理**
图书市场缺货现象时有发生，经典能够经受市场考验。请关注和等待出版社重印或者加印，及时将好书补充到图书馆。

**（六）复本率控制**
图书馆图书的复本率不必太大，甚至可以为"一"，这样能让孩子阅读到更多种类的书。有些图书借阅率非常高，可以适当增加这些图书的复本量。

**（七）班级配置**
建议能给每个年级选择少数优秀图书，配到一个班人手一册（或一个小组一册），方便以后开展相关的阅读活动，同时也能让孩子们有更多共同的话题。

**（八）配套工作**
书籍的配备只是第一步，表示孩子有了阅读的可能。要让孩子爱上阅读，还有很长很长的路要走。要让孩子把书真正读起来，有许多工作要做，例如，高效的图书管理、阅读环境的打造、时间的安排、阅读方法的提示、阅读内容的交流。天图教育基金会愿意与学校和其他机构一起，互相学习，互相借鉴。

**（九）持续更新**
本书目只是基本配备书目，好书还在不断出版，当您发现一本这个书目不容错过的好书的时候，请分享给天图教育基金会，基金会仍会继续潜心研究书目并及时跟踪图书出版市场，将不定期对书目进行修订和发布。

**（十）版权声明**
天图教育基金会为保证本书目的研制，投入了大量人力和物力，为尊重他人劳动，如转引本书目，请注明"本书目由深圳市天图教育基金会研制"字样。
